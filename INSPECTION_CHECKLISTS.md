# Sample Inspection Checklist Templates

This document provides sample inspection checklist templates for different device types.

## Smartphone Inspection Checklist

### Display
- [ ] Screen displays properly (no dead pixels, lines, or discoloration)
- [ ] Touch screen responds accurately across all areas
- [ ] Screen brightness adjusts properly
- [ ] No cracks or scratches on screen

### Audio
- [ ] Earpiece speaker works
- [ ] Loudspeaker works
- [ ] Microphone records clearly
- [ ] Headphone jack works (if applicable)

### Connectivity
- [ ] WiFi connects and functions
- [ ] Bluetooth pairs and connects
- [ ] Cellular signal reception
- [ ] GPS/Location services work

### Battery
- [ ] Battery charges properly
- [ ] Battery holds charge
- [ ] No swelling or damage to battery

### Camera
- [ ] Front camera captures images
- [ ] Rear camera captures images
- [ ] Flash works
- [ ] Camera focus works

### Buttons & Ports
- [ ] Power button functions
- [ ] Volume buttons function
- [ ] Home button functions (if applicable)
- [ ] Charging port works
- [ ] SIM card tray opens/closes

### Sensors
- [ ] Proximity sensor works
- [ ] Ambient light sensor works
- [ ] Fingerprint sensor works (if applicable)
- [ ] Face ID works (if applicable)

### Physical Condition
- [ ] No physical damage to body
- [ ] No water damage indicators triggered
- [ ] All screws present and tight
- [ ] Back cover secure

## Laptop Inspection Checklist

### Display
- [ ] Screen displays properly
- [ ] No dead pixels or backlight issues
- [ ] Hinges open/close smoothly
- [ ] No cracks or damage

### Keyboard & Trackpad
- [ ] All keys function properly
- [ ] Trackpad responds accurately
- [ ] Trackpad buttons click properly

### Ports & Connectivity
- [ ] USB ports work
- [ ] HDMI/Display port works
- [ ] Audio jack works
- [ ] WiFi connects
- [ ] Bluetooth works

### Battery & Power
- [ ] Battery charges
- [ ] Battery holds charge
- [ ] Power adapter works
- [ ] Power button functions

### Performance
- [ ] System boots properly
- [ ] No unusual noises (fan, hard drive)
- [ ] Runs without overheating

### Physical Condition
- [ ] No cracks or dents
- [ ] All rubber feet present
- [ ] Bottom cover secure
